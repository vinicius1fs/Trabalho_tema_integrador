Quais as principais tecnologias, linguagens e ferramentas utilizadas?

As principais linguagens usadas são:

  • HTML - É a estrutura do site. O HTML, ou HyperText Markup Language, é usado para estruturar e organizar o conteúdo de uma página da web. Ele fornece os elementos básicos, como cabeçalhos, parágrafos, imagens e links, que compõem a estrutura de uma página;
  • CSS - É a parte visual do site. O CSS, ou Cascading Style Sheets, é usado para controlar a aparência e o estilo dos elementos em uma página da web. Com o CSS, é possível definir cores, fontes, tamanhos, espaçamentos e outros aspectos visuais. O CSS permite que os desenvolvedores criem layouts responsivos e personalizados, tornando as páginas da web visualmente atraentes e esteticamente agradáveis;
  • Javascript - É a interação do site. Com sua capacidade de criar interações dinâmicas e funcionais em páginas da web, o JavaScript se tornou a base para muitos frameworks e bibliotecas front-end, como o React, Angular e Vue.js. Além disso, o JavaScript é suportado por todos os principais navegadores e é uma habilidade essencial para qualquer desenvolvedor front-end.

As principais ferramentas:

  • Visual Studio Code: Editor de código-fonte gratuito, leve e altamente extensível, usado por milhões de desenvolvedores para programar em HTML, CSS, JavaScript e outras linguagens;
  • Dreamweaver: Editor de código-fonte pago, da adobe;
  • Notepad++: Editor de código-fonte gratuito. É uma versão melhorada do bloco de notas;
  • Git + GitHub: Controle de versão (Git) e hospedagem de código (GitHub) são fundamentais para versionar, colaborar e compartilhar projetos de front-end;
  • DevTools do Navegador: Conjunto de ferramentas integradas nos navegadores (como Chrome, Firefox e Edge) para inspecionar elementos, debugar código e medir performance;
  • Figma: Ferramenta de design de interfaces e prototipação colaborativa usada em time antes do desenvolvimento visual (HTML/CSS);
  • Bootstrap: Framework front-end com componentes prontos para construir layouts responsivos e interfaces modernas com agilidade.

As principais tecnologias:

  • React.js - É uma biblioteca JavaScript de código aberto, mantida pelo Facebook, que permite a criação de interfaces de usuário reativas e eficientes. Com o React, os desenvolvedores podem criar componentes reutilizáveis e construir aplicativos web escaláveis. O React também é amplamente utilizado em desenvolvimento mobile com o React Native. Com uma comunidade ativa e uma vasta seleção de bibliotecas e ferramentas complementares, o React é uma escolha popular para o desenvolvimento front-end;
  • Vue.js - É um framework JavaScript progressivo, conhecido por sua facilidade de uso e curva de aprendizado suave. Ele permite a construção de interfaces de usuário interativas e reativas, tornando-o uma escolha popular entre os desenvolvedores front-end. O Vue.js oferece uma abordagem baseada em componentes, o que facilita a reutilização de código e a manutenção de projetos complexos. Com uma comunidade ativa e uma documentação abrangente, o Vue.js continua ganhando popularidade no cenário do desenvolvimento front-end;
  • Sass - É uma linguagem de folhas de estilo em cascata (CSS) com recursos estendidos. Ele permite que os desenvolvedores escrevam CSS de forma mais eficiente e organizada, com recursos como variáveis, funções e mixins. O Sass é compilado em CSS antes de ser enviado para o navegador, o que facilita a manutenção e a escalabilidade de projetos front-end. Com sua sintaxe amigável e recursos poderosos, o Sass é amplamente utilizado em projetos front-end modernos;
  • TypeScript - Embora possa não ser tão amplamente utilizado quanto o JavaScript puro, o TypeScript está ganhando popularidade rapidamente no mundo do desenvolvimento front-end. Desenvolvido pela Microsoft, o TypeScript é uma linguagem que estende o JavaScript, adicionando recursos de tipagem estática e suporte a recursos avançados de programação orientada a objetos. O TypeScript traz benefícios como a detecção de erros em tempo de compilação e a melhoria da manutenibilidade e escalabilidade de projetos front-end. Muitos frameworks e bibliotecas populares, como o Angular, adotaram o TypeScript como sua linguagem principal;
  • Angular - É um framework JavaScript desenvolvido pelo Google, criado para facilitar o desenvolvimento de aplicações web dinâmicas — especialmente as chamadas Single Page Applications (SPAs).
